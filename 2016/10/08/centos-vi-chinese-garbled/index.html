解决CentOS VI中文乱码问题

解决CentOS VI中文乱码问题

2016-10-08
  • linux

centos 7.1

cd /etc/
vi virc
在最后面增加如下字段:

set fileencodings=utf-8,gb2312,gbk,gb18030  

set termencoding=utf-8  

set fileformats=unix  

set encoding=prc

Over.
